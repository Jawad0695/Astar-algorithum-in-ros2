# Astar-algorithum-in-ros2
his project implements the A* (A-star) path planning algorithm in ROS 2 for autonomous robot navigation in simulated environments. The planner computes an optimal, collision-free path between a start and a goal position using map data and heuristic-based cost evaluation.
